# Vinted clone (frontend)

Backend: https://github.com/Zebiphire/vinted-clone-backend

Preview: https://vinted-clone-frontend.netlify.app/

Features : List offers, Publish offer, Pagination, Search from filters, Pay item, Signup, Login
with: 
- Stripe (https://stripe.com/)
- FontAwesome
- Axios
- BrowserRouter, Navigation, Link
- useState, useEffect
- Cookies

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
